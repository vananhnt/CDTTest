16.70 TRI: TeX REDUCE interface

This package provides facilities written in REDUCE-Lisp for typesetting REDUCE formulas using TEX. The TEX-REDUCE-Interface incorporates three levels of TEXoutput: without line breaking, with line breaking, and with line breaking plus indentation.

Author: Werner Antweiler.
